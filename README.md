# Spring Boot Role based and JWT 🔐

A Spring Boot project showcasing how to implement secure JWT-based authentication and role-based authorization. This repository demonstrates best practices for securing REST APIs with JSON Web Tokens (JWT) and managing user roles effectively, with automated CI/CD packaging.
---

## Implementation
- Creating REST API's
- JWT token generation and validation
- Role-based access control 
- Token expiration handling

---
## STACK
- Java 21+
- Spring boot
- Spring Security
- JWT dependencies
- MYSQL
